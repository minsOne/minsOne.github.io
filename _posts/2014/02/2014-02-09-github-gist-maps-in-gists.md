## Gist안에 Map

Gist는 [GitHub.com 처럼 매핑 GeoJSON 파일](https://help.github.com/articles/mapping-geojson-files-on-github)을 지원합니다. 또한 지도는 내장된 Gist에 표시되어서 쉽게 공유하고 지도를 내장할 수 있습니다!
